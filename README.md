# Multiplier Runner MVP

## Project description
Multiplier Runner is a tiny 2D browser game MVP built with pure HTML, CSS, and JavaScript.  
You start the run, watch an object move smoothly across the game area, and let the multiplier/progress grow over time before stopping to lock your final result.

## How to run
Open `index.html` directly in any modern browser.

## Features
- Start, Stop, and Restart controls
- Smooth movement using `requestAnimationFrame`
- Real-time multiplier and progress updates
- Running/stopped game state display
- Final result shown after stopping
- Responsive layout with touch-friendly buttons for mobile
